Gmail 사용 시

  • 2단계 인증을 활성화한 뒤 앱 비밀번호를 생성하세요.
  • 호스트: smtp.gmail.com, 포트: 587, 암호화: STARTTLS

Microsoft 365 / Outlook

  • 호스트: smtp.office365.com, 포트: 587, 암호화: STARTTLS
  • 관리자가 SMTP AUTH를 허용해야 합니다.

사내 메일 서버

  • IT 담당자에게 SMTP 호스트, 포트, 인증 방식을 확인하세요.
  • 방화벽에서 해당 포트가 열려 있어야 합니다.
  • 내부 릴레이 서버인 경우 고급 설정에서 "SMTP 인증 사용"을 해제하세요.
  • 자체서명 인증서를 사용하면 "SSL 인증서 검증"을 해제해야 연결됩니다.
