인증관리

다중 인증, SMS 발송, 사내 OTP 서버 연동 설정을 구성합니다.

사용자 화면권한 인증관리 보안관리 세션관리 메일관리 품질유형 통합로그 문구관리 버전관리 페이지관리 브랜드관리

다중 인증(MFA)

로그인 직후 두 번째 검증 수단을 요구해 계정 탈취를 방지합니다.

인증 방식 활성화

로그인 시 사용자에게 제공할 인증 방식을 개별적으로 활성화하세요.

OTP 인증 (TOTP)

키를 생성하면 QR코드가 표시됩니다.

Google Authenticator 등 TOTP 앱으로 QR코드를 스캔하세요.

SMS 인증

로그인한 사용자(org_user)의 휴대번호(mobile_phone)로 인증 코드를 발송합니다.

이메일 인증

로그인한 사용자(org_user)의 이메일(email)로 인증 코드를 발송합니다.

사내 OTP 인증

사내 OTP 서버(이니텍/드림시큐리티/미래테크)와 연동하여 인증합니다. 아래 사내 OTP 설정 카드에서 연동 정보를 입력하세요.

MFA가 비활성화되어 있습니다.

SMS 발송 설정

MFA SMS 인증에 사용할 CoolSMS API 자격증명과 발신번호를 입력합니다.

미설정

SMS 발송이 설정되지 않았습니다.

사내 OTP 서버 설정

사내 OTP 솔루션(이니텍·드림시큐리티·미래테크) 연동 정보를 입력합니다.

미설정

사내 OTP 서버가 설정되지 않았습니다.

MFA 확인

선택한 두 번째 인증 수단을 입력하세요.

인증 앱에 표시된 6자리 숫자를 입력하세요.

SMS 테스트 발송

인증코드가 정상 수신되는지 확인합니다.

테스트 코드 123456이 발송됩니다.

사내 OTP 테스트 인증

사내 OTP 서버 연동이 정상인지 확인합니다.

사내 OTP 토큰/앱에 표시된 코드를 입력하세요.
