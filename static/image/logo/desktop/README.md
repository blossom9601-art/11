# Blossom Chat Icon Assets

- `blossom-chat-icon-transparent.png`: transparent master PNG.
- `png/`: common PNG sizes from 16px to 1024px.
- `windows/blossom-chat.ico`: multi-size Windows ICO.
- `macos/BlossomChat.iconset/`: macOS iconset source folder. On macOS, convert with:

```bash
iconutil -c icns BlossomChat.iconset
```

Run that command from inside the `macos` folder to create `BlossomChat.icns`.
