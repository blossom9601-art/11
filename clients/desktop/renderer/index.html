                일반

                Blossom Chat을 어떻게 실행하고 표시할지 제어합니다.

                자동 실행
                Windows 로그인 시 앱이 자동으로 시작됩니다.
                트레이로 최소화
                창을 닫으면 하단 트레이로 숨깁니다.
                언어
                앱에 표시되는 기본 언어입니다.

                알림

                채널·DM·멘션에 대한 푸시, 소리, 방해금지 규칙을 설정합니다.

                전체 알림
                Blossom 알림(토스트/배지) 총괄 on/off
                멘션 알림
                DM 알림
                특정 채널 알림

                키워드

                쉼표로 구분 (예: 긴급, 배포, outage)

                방해 금지

                일정
                이 시간대에는 OS 알림·소리를 끕니다(앱 내 저장).
                ~
                알림 사운드
                포커스 중에도 알림
                Blossom 창이 앞에 있을 때도 데스크톱 알림을 띄웁니다.

                화면

                테마, 가독성, 레이아웃 밀도를 조정합니다.

                모드

                글자 크기
                채팅 밀도
                메시지 줄 간격
                UI 애니메이션

                보안

                접근 제한, 암호화, 감사를 구성합니다. 민감 항목은 관리자 정책이 우선될 수 있습니다.

                접근

                앱 잠금 PIN 정책
                채팅 잠금 화면에 PIN을 사용 (별도 OS 보안 영역에 저장)
                자동 잠금 시간(분)
                파일 다운로드 제한
                허용 파일 형식
                쉼표로 구분합니다. 예: pdf, png, jpg, docx, zip
                파일 1개당 최대 용량(MB)
                대화창 파일 첨부 전에 로컬에서 차단합니다.
                복사 제한(클립보드)

                엔터프라이즈 보안

                로컬에서 가능한 수준의 보강·기록을 설정합니다(일부는 서버·정책과 연동).

                TLS·인증서 검증 활성
                인증서·호스트 Pinning(신뢰)
                Refresh token rotation(기대)
                로컬 토큰/비밀번호 암호화 저장 OS
                감사 로그(로컬)
                설정 변경 이력(로컬)

                대화 자동삭제 정책

                마지막 대화 이후 설정한 시간 동안 새 메시지가 없으면 대화 내용이 자동 삭제됩니다. 관리자는 전역, 일반 사용자는 내 채팅에만 반영됩니다.

                현재 계정 범위로 자동삭제 정책을 저장합니다.
                자동삭제 활성화
                비활성 상태가 보관기간을 넘으면 메시지를 soft delete합니다.
                보관기간
                대화가 이어지면 삭제 시간 리셋
                마지막 메시지 이후 설정한 시간이 지나면 대화 내용이 삭제됩니다.
                첨부파일 함께 삭제
                기존 대화방에도 적용

                연결

                Blossom 서버와의 통신, 프록시, 정책을 관리합니다.

                서버 주소 관리자
                인증서·TLS 상태

                프록시

                관리자 정책(서버 잠금) 표시
                활성화 시 위 서버/프록시 일부는 정책에 의해 잠깁니다(키: adminPolicyServerLock).

                데이터

                캐시·로그·로컬 저장소를 정리합니다. 이 작업은 복구가 어려울 수 있습니다.

                캐시 삭제
                다운로드 데이터
                로그(로컬) 삭제
                로컬 앱 데이터/세션 초기화

                초기화는 이 PC에 저장된 접속 정보·쿠키·캐시·임시 데이터를 지우고 앱을 다시 시작합니다. 채팅 데이터와 계정 권한은 중앙 서버에서 다시 가져옵니다.
