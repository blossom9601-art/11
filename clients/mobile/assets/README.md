# 모바일 앱 자산

배포 빌드 시 다음 PNG 파일을 추가하세요:

| 파일 | 용도 | 권장 크기 |
|------|------|-----------|
| `icon.png` | 앱 아이콘 | 1024×1024 |
| `splash.png` | 스플래시 화면 | 1284×2778 |
| `adaptive-icon.png` | Android 적응형 아이콘 (foreground) | 1024×1024 |
| `notification-icon.png` | Android 알림 아이콘 (단색) | 96×96 |

자산 추가 후 `app.json`의 해당 필드를 다시 활성화하세요.
